HTML Introduction

→HTML stands for HyperText Markup Language.It is the standard markup language used to create and structure content on the World Wide Web (WWW).
→It is used to create web pages and web applications.
H→Hyper
T→Text
M→Markup
L→Language
Hyper Text :
Is the text used to link various web pages
Mark-Up :
It highlights text either by underlines or displaying it with an different colors, or both
Language :
It refers to the way of communication between web pages, which has its own syntax and rules
→HTML is the backbone of most web pages and is used to define the structure and layout of web documents, including text, images, links, forms, and multimedia.

HTML Tags

👉 !DOCTYPE :

The tag in HTML is used to declare the document type and version of HTML being used. It helps web browsers to understand how to interpret the HTML markup in the document, ensuring proper rendering and compatibility with different standards.

👉 Head tag :

We will use the head tag for the page title.
Also linking the CSS, js will also be linked in the head tag itself..

👉 Title tag :

Suppose when we open any Website it has a title we give it in the title tag

👉 Body tag :

All the code we write is written in the body tag only.

👉 Heading tags :

→ There are 6 heading tags.
→ H1 to H6
→ Heading tags will be in decreasing order.

👉 P tag - Paragraph tag :

→used to define paragraphs of text.
→Its a block element, meaning it creates a new line before and after it's content.
→Tts commonly used to structure text content on web pages, separating paragraphs visually.

👉 Div tag :

→it is used as a generic container for grouping and structuring content.
→its a block level element, providing a way to create divisions or sections within a webpage.

👉 Span tag :

→it is used to apply styling or manipulate specific portions of text within a larger block of content.
→its an inline element, meaning it doesn't force line breaks before or after its content.

👉 Comment tag :

→ In HTML, the comment tag allows you to insert comments in the code that won't be displayed in the browser.
→ Its useful for adding notes to your code for yourself or other developers to understand it better.
→ we use < !-- to start a comment and --> to end a comment.
→ Anything in between won't be rendered on the webpage

👉 Anchor tag :

→ Hyperlinks in html commonly known as anchor tags.
→ It allows you to create clickable links that direct users to other web pages,files,or sections within the same page.
→ Hyperlinks can also include text or images which users can click on to navigate to the linked content.

👉 Bold tag - b tag :

→ In HTML, the b tag is used to make text bold.
→ When we wrap text with < b> and < /b> tags the enclosed text will appear bold in the browser.
→ It's a simple way to add a emphasis or highlight certain parts of your content.
→ It is a presentational tag.

👉 Italic tag - I tag :

→The I tag in html Is used to italicize text.
→ When you enclose text within < i> and < /i> tags, the enclosed text will appear in italics in the browser.
→ It is a presentational tag.

👉 Underline tag - u tag :

→ In HTML, the < u> tag is used to underline text.
→ When you enclose text within < u> and < /u> tags, the enclosed text will appear underlined in the browser.
→ We can say that the u tag is considered deprecated in HTML5, means it's not recommended for use in modern web development.

👉 Img tag :

→ The img tag in html is used to embed images into a webpage.
→ It is a self closing tag, means it doesn't have a closing tag.
→ Images are not technically inserted into a web page, images are linked to web pages.
→ The img tag creates a holding space for the referenced image.

👉 Video tag :

→ The video tag in html is used to embed videos into a webpage. → It allows you to play videos directly within the browser without requiring any additional plugins.

👉 Sub tag - subscript tag :

→ In HTML, the sub tag is used to define subscript text.
→ Subscript text appears slightly lower and smaller than the normal text
→ It's typically used for mathematical and chemical formulas, footnotes or indicating abbreviations.

👉 Sup tag - superscript tag :

→ In HTML, the sup tag is used to define superscript text.
→ Superscript text appears slightly higher and smaller than the normal text.
→ It's typically used for mathematical expressions, exponents, footnotes or indicating abbreviations.

👉 Marqee tag :

→The marquee tag in html is used to create a scrolling text or image effect within a webpage.
→ It's a non standard HTML element.
→ It's usage is discouraged because it's considered outdated and not supported in modern web standards.

👉 Abbr tag :

→ In HTML, the abbr tag is used to define the abbreviation or acronym.
→ it's used to shortened versions and used to represent a series of letter.
→The abbreviation is used to provide the useful information to browsers, translation systems and search engines.

👉 Brake tag - Br tag :

→The br tag in html is used to insert a line break, means it creates a new line within the text or content.
→ It doesn't have a closing tag because it's a self closing tag.
→ It's commonly used to separate lines of text or creating spacing within a document.

👉 Hr tag - horizontal rule tag :

→The < hr> tag in HTML is used to create a horizontal line, often referred to as a "horizontal rule.
→It's a self-closing tag, meaning it doesn't have a closing tag.
→ The < hr> tag is commonly used to visually separate content within a webpage, such as sections of text, images, or other elements.

👉 Bdo tag - Bi- directional override tag :

It is used to override the current text direction or to change the text direction.

👉 Big tag :

→The big tag is used to increase the size of the text enclosed within it.
→Its not recommended to use because it's presentation can vary across different browsers and it doesn't provide semantic meaning.
→ so better to use css for the styling text size instead.

👉 Center tag :

→It was used to set the alignment of text into the center.
→ This tag is not supported in HTML5.
→ CSS is used to set the alignment of text instead of the center tag in HTML5.

👉 Cite tag :

→ The cite tag is used to define the title of a creative work, such as a book, a movie etc.
→ It's typically used to enclose the title of the work being referenced within a block of text.

👉 Header tag :

→In HTML, the < header> tag is used to define introductory or navigational content at the beginning of a section or a page.
→ It typically contains headings, logos, navigation menus, search bars, and other introductory elements.
→ The < header> tag is commonly used within the < body> element to structure the layout of a webpage and provide meaningful organization to its content.

👉 Html tag :

→ The html tag is the root element of an HTML document.
→It wraps all the content of the document, including the < head> and < body> sections.
→It defines the beginning and end of an HTML document and contains metadata about the document, such as the character encoding and language.
→All other HTML elements must be nested within the < html> tag.

👉 Footer tag :

→The < footer> tag in HTML is used to define the footer of a document or a section.
→It typically contains information about the author, copyright notices, contact information, links to related documents, or other auxiliary information.
→The < footer> tag is commonly used within the < body> element to provide structural organization to the content of a webpage, typically appearing at the bottom of the page or at the end of a section.

👉 Iframe tag :

→The < iframe> tag in HTML is used to embed another HTML document within the current document.
→ It stands for "inline frame.
→This allows you to display content from another web page or an external source, such as a video, map, or social media feed, directly within your own webpage.

👉 Nav tag :

→The < nav> tag in HTML is used to define a section of navigation links within a document.
→It's typically used to wrap around lists of links that navigate to different parts of the website or sections within the webpage.

👉 Q tag :

→The < q> tag in HTML is used to indicate a short inline quotation.
→ When you enclose text within the < q> tag, browsers typically render it with quotation marks.
→It's useful for marking up short quotations within a paragraph or sentence.

👉 Strong tag :

→The < strong> tag in HTML is used to indicate that the enclosed text should be rendered with strong emphasis, typically displayed as bold text by browsers.
→It's used to convey importance, seriousness, or relevance within the content.

LIST TAGS :

👉 Li tag - list tags :

→The < li> tag in HTML is used to define a list item within an ordered list (< ol>) or an unordered list (< ul>).
→ It represents each individual item in the list. →The < li> tag is a block-level element by default, meaning each list item typically starts on a new line.

👉 Ol tag - ordered list tags :

→The < ol> tag in HTML is used to create an ordered list.
→An ordered list is a list of items where each item is numbered or ordered sequentially.
→Each item in the list is defined by the < li> (list item) tag, which is nested within the < ol> tag.

👉 Ul tag - unordered list tags :

→The < ul> tag in HTML is used to create an unordered list.
→An unordered list is a list of items where the order of the items does not explicitly matter.
→Each item in the list is defined by the < li> (list item) tag, which is nested within the < ul> tag.

TABLE TAGS :

👉 Table tag :

→The < table> tag in HTML is used to create tables on web pages.
→ It allows you to organize data into rows and columns.
→Inside the < table> tag, you use additional tags like < tr> for rows, < td> for data cells, and < th> for header cells.

👉 Th tag - table header tag :

→The < th> tag in HTML is used to define header cells in a table.
→Header cells are typically used to label the columns or rows of a table.

👉 Td tag - table data tag :

→ The < td> tag in HTML is used to define data cells in a table.
→It represents individual cells within a row and contains the actual data that is displayed in the table.

👉 Tr tag - table row tag :

→The < tr> tag in HTML is used to define a row within a table.
→ It stands for "table row" and contains one or more elements, which represent the individual cells within that row.

👉 Rowspan tag :

→The rowspan attribute in HTML is used within a < td> or < th> tag to specify how many rows a cell should span vertically in a table.
→ This allows a cell to occupy multiple rows, effectively merging them into a single larger cell.

👉 Colspan tag :

→The colspan attribute in HTML is used within a < td> or < th> tag to specify how many columns a cell should span horizontally in a table.
→ This attribute allows a cell to stretch across multiple columns, merging them into a single larger cell within the table.

FORM TAGS :

👉 Label tag :

→ The tag in HTML is used to define a label for an < input>, < select>, < textarea>, or < button> element.
→ It provides a textual description or title for the associated form control, improving accessibility and user experience.
→ When you click on a label, it focuses or activates the associated form control.

👉 Input tag :

→The < input> tag in HTML is used to create various types of input fields on web forms.
→ It allows users to input data, such as text, numbers, checkboxes, radio buttons, and more.

👉 Textarea tag :

→ The < textarea> tag in HTML is used to create a multiline text input field in web forms.
→ Unlike the single-line < input> field, < textarea> allows users to input multiple lines of text.
→ It's commonly used for larger text inputs such as comments, messages, or descriptions.
→ The size of the textarea can be adjusted using the rows and cols attributes.

👉 Address tag :

→ The < address> tag in HTML is used to define contact information for the author or owner of a document or article.
→ It is typically used to include the author's name, physical address, email address, or other contact details.

👉 Select tag :

→ The < select> tag in HTML is used to create a dropdown list of options.
→ It allows users to choose one option from a list of predefined choices.
→ Each option is defined within the < select> tag using the < option> tag.

👉 Option tag :

→ The < option> tag in HTML is used within a < select> element to define an individual option in a dropdown list.
→ Each < option> tag represents one item in the list of choices presented to the user.

👉 Type tag :

→ In HTML, the type attribute is used to specify the type of an < input> element. It determines what kind of input field will be rendered on the web page.
→ The type attribute is essential for defining the behavior and appearance of input elements in HTML forms.

👉 Button tag :

→ The < button> tag in HTML is used to create a clickable button on a web page.
→ It can contain text, images, or other HTML elements.
→ It's a versatile element for adding interactive functionality to web pages.

👉 Radio tag :

→ The < input type="radio"> defines a radio button.
→ Radio buttons are normally presented in radio groups (a collection of radio buttons describing a set of related options). Only one radio button in a group can be selected at the same time.

👉 Checkbox tag :

→ The < input type="checkbox"> defines a checkbox.
→ The checkbox is shown as a square box that is ticked (checked) when activated.
→ Checkboxes are used to let a user select one or more options of a limited number of choices.

👉 Submit tag :

→ The < input type="submit"> defines a submit button which submits all form values to a form-handler.
→ The form-handler is typically a server page with a script for processing the input data.
→T he form-handler is specified in the form's action attribute.
